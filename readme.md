
###Dimmer

TRIAC dimming circuit for 12V AC lights.

Controlled by A PIC12F675, opto-isolated zero-crossing detector.

Output is set with a variable resistor.

###Tools

CadSoft Eagle

Microchip MPLAB IDE 7

CCS PICC C-Compiler
